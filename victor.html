This project is aimed to design a web based management system using United Bank of Africa as a case study for the needs of accurate and efficient database application towards effective information retrieval and decision making. The project was designed using Asp.net and part of Php (preprocessor hypertext) and Microsoft Access as the database for storing and retrieving of records. It is used to keep proper recruitment process in areas of inputting new staff record, updating existing staff information, deleting and it provides a solid protection in which each user personnel user (staff) has he/her personal username and password to gain access to its information. All records are saved and can be retrieving from the database.
